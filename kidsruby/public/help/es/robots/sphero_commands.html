Órdenes de Sphero

Regresar a Robots Regresar a Sphero

La siguiente lista de órdenes pueden ser usadas para controlar tu Sphero

Sphero.start(port) do...end

Conecta a Sphero y ejecuta las órdenes que se encuentren entre 'do' y 'end'

port(puerto) - cómo conectar a un Sphero en particular.

roll speed, heading

Hace que tu Sphero ruede en una dirección en particular

speed(velocidad) - un número entre 0 y 255

heading(dirección) - un número entre 0 y 359 que especifica en qué dirección ir

stop

Hace que Sphero se detenga

heading= h

Hace que Sphero cambie su dirección cuando ya está rodando

h - un número entre 0 y 359 que especifica en qué dirección ir

rgb r, g, b

Cambia el color del LED de Sphero

r - un número entre 0 y 255

g - un número entre 0 y 255

b - un número entre 0 y 255

keep_going duration

Le dice a Sphero que se mantenga haciendo lo que sea que ya esté haciendo por cierta cantidad de tiempo

duration(duración) - un número que especifica por cuántos segundos mantener la acción

ping

Hace ping a Sphero para ver si responde

FORWARD

ADELANTE - Un valor constante de 0

BACKWARD

ATRÁS - Un valor constante de 180

RIGHT

DERECHA - Un valor constante de 90

LEFT

IZQUIERDA - Un valor constante de 270

Hay más órdenes, pero estas son las más importantes. Para más información, echa un vistazo a la documentación de la gema Sphero en Github
