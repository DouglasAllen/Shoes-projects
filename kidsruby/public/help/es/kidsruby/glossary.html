Glosario

¡Bienvenido!

¿Qué es un glosario?

Un glosario es una coleccíón de definiciones de palabras que son usadas en un tema en particular. Podría definirse como un diccionario temático.

Glosario

carácter: Una sola letra, número o símbolo

clase: Una clase es una plantilla para crear objetos. Un objeto de una clase dada es llammado 'instancia' de esa clase. Una clase puede contener atributos, métodos e incluso otras clases!

método: Los métodos permiten que las clases realicen acciones. Son una serie de instrucciones y pueden regresar un valor

parámetro: Un tipo de variable entre paréntesis, el cual pone un límite o varios límites en el método en el cual se encuentra.

cadena: Una cadena es un conjunto de caracteres (letras, dígitos, símbolos) encerrados dentro de comillas. Pueden usarse comillas simples o comillas dobles para crear cadenas (Fuente: Why's Poignant Guide)

ícono: Una imagen en una pantalla que representa una archivo específico, directorio, ventana, opción, o programa. (Fuente: The Free Dictionary)

pixel: Cualquiera de un número de elementos de una imagen muy pequeña que hace a la imagen, así como una una unidad de proyección visual (Fuente: The Free Dictionary)

atributo: Una propiedad o característica de una Entidad. Por ejemplo, los atributos de una persona serían el nombre, la edad, la estatura, etc... (Fuente: Computing Students)

variable: Un símbolo usado para guardar datos que se asignan a la variable. Por ejemplo, en x = 50, "x" es la variable y "50" es el dato.

objeto: Un elemento que puede ser seleccionado y maniobrado, como un elemento gráfico en pantalla. Estos objetos pueden incluir datos y los procedimientos necesarios para manejar esa información. (Fuente: The Free Dictionary)

.png: Acrónimo para "Portable Network Graphics" y es un formato de archivo para compresión de imágenes, el cual es sólo un término elegante para reducir el tamaño de una imagen sin reducir mucho la calidad.

instancia: Creando una instancia de una clase. Por ejemplo, creando una instancia de la clase "Cat" podría ser "My Cat".
